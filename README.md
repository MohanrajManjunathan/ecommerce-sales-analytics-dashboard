# E-Commerce Sales Analytics Dashboard

## Project Overview

This project analyzes e-commerce sales data using Python and Power BI to identify revenue drivers, customer behavior, product performance, and sales trends.

## Tools Used

* Python
* Pandas
* NumPy
* Power BI

## Data Cleaning

* Handled missing values
* Standardized category names
* Removed inconsistencies
* Created month and weekday features
* Identified anomalies
* Segmented customers

## Dashboard Pages

### 1. Executive Dashboard

* Total Revenue
* Average Order Value
* Unique Customers
* Return/Cancel Percentage

### 2. Revenue Analysis

* Revenue by Category
* Revenue by Month
* Revenue by Weekday
* Payment Method Analysis

### 3. Product Analysis

* Top Products
* Revenue Contribution
* Product Performance

### 4. Revenue Trend Analysis

* Monthly Trends
* Weekday vs Weekend Revenue
* Best and Worst Months

### 5. Customer Analysis

* Customer Segmentation
* Top Customers
* Revenue Contribution
* Repeat Customer Analysis

### 6. Insights & Recommendations

* Key Business Findings
* Strategic Recommendations

## Key Insights

* Clothing generated the highest category revenue.
* Shoes contributed approximately 20% of total revenue.
* October was the highest revenue month.
* Customer_142 was the highest-value customer.
* Return and cancellation rates exceeded 42%.

## Business Recommendations

* Reduce return and cancellation rates.
* Improve electronics product performance.
* Increase customer retention programs.
* Launch promotions during low-performing months.
* Focus on high-performing products and customer segments.
